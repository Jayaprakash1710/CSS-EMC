  • Element selector
  • class selector
  • Id selector
  • Universal selector

Element Selector:

Class Selector:

In element selector we select tags and apply styles.

If there are multiple same tags, same style would be applied for all the tags. What is we want to apply different styles for different tags with same property? That's why we have class selector.

ID Selector:

  • id is unique. A single id can be used only once in the whole HTML file. It can't be used for anyother tags.
  • But we can have multiple classes. That's the key difference between id and class.
