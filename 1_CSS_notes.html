  • Element selector
  • class selector
  • Id selector
  • Universal selector

Class Selector:

In element selector we select tags and apply styles.

If there are multiple same tags, same style would be applied for all the tags. What is we want to apply different styles for different tags with same property? That's why we have class selector.

CSS Box Model:

The CSS box model is essentially a box that wraps around the HTML element.
It consists of,

  • Margins
  • Borders
  • Padding
  • Content

Display Property:

  • Inline - Same line, fill only the size of the content, cannot set width and height
  • Inline block - Same line, fill only the size og the content, can set width and height
  • Block - separate line, blocks the psace of whole line, can set width and height
  • None - This will hide the element from HTML
